海关对电子与美妆类商品的申报细则做出微调，强调产品描述的具体性与估价一致性。建议卖家复核内部 HS Code 映射与发票字段，避免低报与泛描述导致查验。

## 建议动作
- 检查 HS Code 与发票匹配
- 增补成分 / 材质字段
- 使用自动校验工具标记潜在风险描述
